
# Tic-Tac-Toe Web Application

## Objective

The objective of this project is to build a simple and interactive Tic-Tac-Toe game using frontend technologies. It demonstrates basic concepts of game logic, user interaction, and responsive interface design.

---

## Technologies Used

* HTML for structuring the application
* CSS for styling and layout
* JavaScript for implementing game logic and interactivity

---

## Features Implemented

* Two-player gameplay (Player X and Player O)
* Turn-based system with clear indication of the current player
* Winner detection based on predefined winning combinations
* Draw detection when all cells are filled without a winner
* Scoreboard to track wins and ties
* Reset functionality to clear the board
* Popup message to display game result
* Responsive and clean user interface

---

## Steps to Run the Application

1. Clone the repository:

   ```
   git clone <your-github-repo-link>
   ```

2. Open the project folder.

3. Run the application by opening the `index.html` file in a web browser.
